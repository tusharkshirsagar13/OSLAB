🔹 8. Main Loop

Iterates through each page in reference string
🔹 9. HIT Check
if(frames[j] == ref[i])
➤ If page found:
hit = 1
No replacement required
🔹 10. MISS (Page Fault)

If page is not present in frames

🔹 11. Optimal Replacement Logic
int replace_idx = -1, farthest = i;
replace_idx → index of page to replace
farthest → tracks the farthest future use
🔹 12. Find Page to Replace

Loop through all frames:

➤ Case 1: Empty Frame
if(frames[j] == -1)
Use empty frame immediately
➤ Case 2: Find Future Usage
int next_use = n + 1;
Assume page is never used again

Then check future:

for(int k = i + 1; k < n; k++)
Find next occurrence of current frame page
➤ Select Optimal Page
if(next_use > farthest)
Replace the page which is:
Used farthest in future OR
Not used again at all

👉 This is the core idea of Optimal Algorithm

🔹 13. Replace Page
frames[replace_idx] = ref[i];
faults++;
Insert new page
Increment fault count
🔹 14. Output Each Step
➤ If FAULT:
Ref   Frames   FAULT
➤ If HIT:
Ref   Frames   HIT
Displays memory state after each reference
🔹 15. Final Output
No. of faults : faults
No. of hits   : n - faults
🔹 16. Algorithm Used

✅ Optimal Page Replacement Algorithm

🔹 17. Key Concept of Optimal
Replace the page:
👉 Which will be used farthest in future
👉 OR Never used again
🔹 18. Important Observation
Looks ahead in reference string (future knowledge)
Gives minimum possible page faults
Not practical in real OS (used for comparison)
