🔹 11. FIFO Replacement Logic
frames[front] = pages[i];
front = (front + 1) % f;
Replace page at front
Move pointer circularly using modulo %

👉 This ensures:

Oldest page is replaced first
Queue behavior (First In First Out)
🔹 12. Update Counters
faults++;
On MISS → increment faults
hits++;
On HIT → increment hits
🔹 13. Output Each Step
➤ If FAULT:
Page   Frames   Fault
➤ If HIT:
Page   Frames   Hit
Displays current frame status
🔹 14. Final Output
Total Hits   = hits
Total Faults = faults
🔹 15. Algorithm Used

✅ FIFO (First In First Out) Page Replacement

🔹 16. Key Concept of FIFO
Replace the page:
👉 Which came first (oldest in memory)
🔹 17. Important Observation
Uses circular queue logic
Does NOT consider:
Frequency of use
Recent usage

👉 Can lead to Belady’s Anomaly (more frames → more faults)
