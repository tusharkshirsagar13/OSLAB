1. Header File
#include <stdio.h>
→ Used for input/output (printf, scanf)
🔹 2. Structure Definition
struct Process {
    int id, at, bt, st, et, tat, wt, completed;
};
➤ Fields Meaning:
id → Process ID
at → Arrival Time
bt → Burst Time
st → Start Time
et → End Time (Completion Time)
tat → Turnaround Time
wt → Waiting Time
completed → flag (0 = not done, 1 = done)
🔹 3. Input: Number of Processes
scanf("%d",&n);
User enters total number of processes
🔹 4. Input: AT & BT
scanf("%d %d", &p[i].at,&p[i].bt);
For each process:
Arrival Time
Burst Time
🔹 5. Initialization
p[i].completed = 0;
Marks all processes as not executed
🔹 6. Algorithm Used

✅ SJF (Shortest Job First – Non-Preemptive)

🔹 7. Main Loop
while(count < n)
Runs until all processes are completed
🔹 8. Find Shortest Job
int idx = -1;
int min_bt = 1e9;
idx → selected process index
min_bt → smallest burst time
➤ Selection Condition
if(p[i].at <= current_time && !p[i].completed)
Process must:
Have arrived
Not completed
if(p[i].bt < min_bt)
Select process with smallest burst time
🔹 9. CPU Idle Condition
if(idx == -1){
    current_time++;
}
If no process available → CPU waits
🔹 10. Process Execution
p[idx].st = current_time;
Start Time
p[idx].et = p[idx].st + p[idx].bt;
End Time
🔹 11. Time Calculations
➤ Turnaround Time:

TAT=ET−AT

p[idx].tat = p[idx].et - p[idx].at;
➤ Waiting Time:

WT=TAT−BT

p[idx].wt = p[idx].tat - p[idx].bt;
🔹 12. Update Time
current_time = p[idx].et;
Move CPU forward
🔹 13. Mark Process Completed
p[idx].completed = 1;
count++;
🔹 14. Output Table
ID   AT   BT   ST   ET   TAT   WT
Displays scheduling results
🔹 15. Average Calculation
total_tat += p[i].tat;
total_wt += p[i].wt;
Compute:
Average TAT = total_tat / n  
Average WT  = total_wt / n
🔹 16. Key Concept of SJF
Select process with:
👉 Shortest Burst Time
Once started → runs till completion
🔹 17. Important Observations
Gives minimum average waiting time (optimal)
Non-preemptive → no interruption
Can cause starvation:
→ Long processes may wait
🔹 18. Example Flow

Processes:

P1: AT=0, BT=6  
P2: AT=1, BT=2  
P3: AT=2, BT=4  

Execution:
👉 P1 → P2 → P3
(based on arrival + shortest job among available)

🔹 19. Time Complexity
For each process → scan all

👉 O(n²)
