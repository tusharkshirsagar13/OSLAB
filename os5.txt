🔹 2. Structure Definition
struct Process {
    int id, at, bt, rt, st, et, tat, wt;
    int started;
};
➤ Fields Meaning:
id → Process ID
at → Arrival Time
bt → Burst Time
rt → Remaining Time
st → Start Time
et → End Time
tat → Turnaround Time
wt → Waiting Time
started → flag to check if process started

🔹 17. Key Concept of SRTF
At every time unit:
👉 Pick process with shortest remaining time
Can preempt running process
🔹 18. Important Observations

Very dynamic scheduling
